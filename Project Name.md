# 名称

**Deutzia**

**溲疏属Deutzia**隶属于虎耳草科，全世界有60个种左右，分布在亚州和中美洲，观赏期集中在春末夏初，一般在花园中属于边界植物。

以60种溲疏作为Project Name，同时吉祥物和贴图用实物，图标颜色也可改为相应颜色。

## 花语

溲疏花的花语：忠实、可信任的
